AI Multi-Agent Research System

This project is an AI-powered research assistant that automates the entire process of gathering, analyzing, and delivering insights on any given topic. It combines multiple specialized AI agents into a single workflow that plans research, retrieves information from the web, synthesizes findings into a structured report, and delivers the results via email.

Key Capabilities

  • Automated research planning: Breaks down complex queries into targeted search actions
  • Parallel information retrieval: Executes multiple web searches concurrently for speed and efficiency
  • Intelligent synthesis: Generates detailed, structured reports (1000+ words) with summaries and follow-up questions
  • End-to-end automation: Delivers results as a formatted HTML email
  • Real-time user experience: Streams progress updates during execution

How It Works

The system orchestrates a pipeline of AI agents, each responsible for a specific task:

  1. Planner Agent – analyzes the query and defines a research strategy
  2. Search Agent – gathers relevant information using web search tools
  3. Writer Agent – synthesizes results into a comprehensive report
  4. Email Agent – formats and sends the final output

Technical Highlights

  • Multi-agent architecture with clear separation of responsibilities
  • Asynchronous execution with parallel task processing (Python asyncio)
  • Structured outputs using typed schemas (Pydantic)
  • Tool-augmented AI agents (web search, email API integration)
  • Streaming responses for improved user experience (Gradio interface)
  • Traceability and observability of agent workflows

Technologies Used

Python, Asyncio, Gradio, OpenAI Agents SDK, Pydantic, SendGrid API

Impact

This project demonstrates how AI can move beyond simple chat interfaces into autonomous, goal-oriented systems. It showcases practical implementation of AI agents working together to solve real-world tasks, highlighting skills in AI architecture, orchestration, and product-oriented design.
